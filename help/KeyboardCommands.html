Versatile Radio Programmer (VRP) — Keyboard Commands

Every command in VRP, and the key that runs it. On Windows, menu commands use Ctrl; on macOS the same commands use Command. The arrow keys, Space, Tab, Enter, Escape, and the function keys are the same on both.

Press F1 in the app for a quick version of this list without leaving the program. For a walkthrough of what these commands do, see the Getting Started guide.
